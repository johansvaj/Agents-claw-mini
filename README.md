# рҹҺҜ Usage

Create Agent

```bash

```

Run Agent

```bash
nexcorix run --agent coder
```

List Agents

```bash
nexcorix list-agents
```

Memory Sync

```bash
nexcorix memory sync
```

Memory Search

```bash
nexcorix memory search "project"
```

MCP Status

```bash
nexcorix mcp status
```
